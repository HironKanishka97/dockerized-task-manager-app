# SimpleTask Full-Stack Application

This repository contains the **backend server** and database setup for the SimpleTask application.  **It uses **Spring Boot** for the backend and **MySql** as the database and "React +vite** for the frontend.  

With Docker and Docker Compose, you can run the full application locally with a single command.

---

## Prerequisites

Before running the application, make sure you have:

- [Docker](https://www.docker.com/get-started) installed  
- [Docker Compose](https://docs.docker.com/compose/install/) installed  

---

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/HironKanishka97/dockerized-task-manager-app.git
cd simpletask 
```
2. **Run docker compose file**

```bash
docker compose up --build
```

3. **Now the app will start and will run on http://localhost:5174/**
